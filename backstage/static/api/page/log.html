Log

Rouwan的Log基本提供了3类日志记录, 分别是system(系统), error(错误), backstage(后台), Rouwan同样也支持自定义日志
关于 rw.log 的更多信息请参考左侧 API 内的 system.log

自定义日志配置举例

首先配置Config内的Log信息 
log:{
	test:{    /* 日志名称 */
		color:32,    /* 颜色(PS1) */
		filename:'system',    /* 日志文件名前缀 */
		path:'./log/',    /* 日志存储地址 */
		of:true    /* 是否开启该日志记录 */
	},
	/* ...省略... */
颜色代码请参考: https://wiki.archlinux.org/index.php/Color_Bash_Prompt

实际应用举例

使用如上的配置, 记录一个 test 的日志 
rw.log.write('This is a log.','test');
该操作将会创建一个类似 ./log/test_20140326.log 的文件, 记录日志产生日期及内容, 并且在控制台打印
关于 rw.log 等信息的更多资料, 请参考左侧 API 内的 system.log
