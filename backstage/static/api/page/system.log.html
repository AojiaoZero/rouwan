rw.log.write ( data:string, type:string, sync:boolean = false ):null

记录一段日志

data 为 string ，日志内容
type 为 string ，日志的类型，对应配置文件中的内容
sync 为 boolean ，是否同步写入日志文件；若为 true ，则同步写入日志文件；若为 false ，则异步写入日志文件
 返回 null 
/* 举例 */
rw.log.write('This is a log.', 'test')
