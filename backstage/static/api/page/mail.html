Mail

Rouwan的Mail功能使用 nodemailer
更多信息请参考左侧 API 内的 system.mail

配置举例

首先配置Config内的Mail信息 
mail:{
	'test':{    /* mail服务的名字，自定义 */
		service:'Gmail',    /* 所使用邮箱的服务商，支持情况请参考本页下方的"支持的服务商" */
		auth:{
			user:"user",    /* 邮箱用户名 */
			pass:"pass"    /* 邮箱密码 */
		}
	},
	/* ...省略... */

实际应用举例

使用配置信息为 test 的账户发送邮件给 to@example.com 
rw.mail('test',{
	from: "test <test@example.com>",    /* 发件人 */
	to: "to@example.com",    /* 收件人 */
	subject: "subject",    /* 邮件标题 */
	html:"content"    /* 邮件内容 */
	},
	function(error,data){
		/* ...省略... */
	}
);
