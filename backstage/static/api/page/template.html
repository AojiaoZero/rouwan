Template

Rouwan提供了简单的模板和模板缓存功能
关于 rw.t 的更多信息请参考左侧 API 内的 system.template
模板文件修改后，Rouwan会自动在一定时间(默认为5s)后重新加载修改过的新模板并刷新缓存，无需重启Rouwan

渲染模板举例

模板内容，该模板保存为 /www/template/name.html 
<div>My name is {{NAME}}.</div>
渲染该模板 
rw.t.render('/www/template/name.html',{'NAME':'Sai'});
输出结果 
<div>My name is Sai.</div>
执行一次后， /www/template/name.html 会被加入缓存，第二次调用时将直接从缓存中读取

功能

Rouwan暂时只支持单纯的变量替换的模板渲染，替换格式为HTML中 {{VAR}} 替换为渲染变量中的 {'VAR':'VALUE'}
