配置

配置文件位于该位置 
./node_modules/rouwan/config/config.example.js
该配置文件为配置参考，请复制该文件为 config.js 以使用

Config.js

全局访问方式: rw.config

启动脚本

使用 nohup 后台启动 run.js 的 sh 脚本，并输出控制台内容到 nohup.log 
// start
nohup node run.js > ./hup.log &
