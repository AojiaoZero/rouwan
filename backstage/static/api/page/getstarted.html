开始

首先请配置配置文件，完成配置vHost等信息后，可以通过以下方式启动Rouwan
点此获得帮助：Config - 关于配置

使用Rouwan，只需简单两行代码。 
require('rouwan');	/* 加载Rouwan */
rw.server.start();	/* 启动Rouwan */

使用Session/MongoDB

如需使用Session/MongoDB, 请在启动Rouwan前完成Session/MongoDB的初始化 
require('rouwan');	/* 加载Rouwan */
rw.sess.init(function(){	/* 初始化Session, 完成后执行Callback */
	rw.mongo.init(function(){	/* 初始化MongoDB, 完成后执行Callback */
		rw.server.start();	/* 启动Rouwan */
	});
});
